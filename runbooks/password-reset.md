# Password Reset Procedure

**Category:** Account Management  
**Tier:** 1  
**Tools:** Active Directory / Help Desk Ticketing System

---

## When to Use This
User is locked out, forgot their password, or password has expired.

---

## Steps

1. Verify the user's identity before making any changes
   - Ask for employee ID, email, or manager's name
2. Open Active Directory Users and Computers
3. Search for the user by name or username
4. Right-click the account → **Reset Password**
5. Set a temporary password following company policy
   - Usually requires uppercase, number, special character
6. Check **"User must change password at next logon"**
7. If account is locked out, check **"Unlock account"**
8. Inform the user of their temporary password securely
9. Document the action in the ticket and close it

---

## Notes
- Never reset a password without verifying identity first
- Always use a different temp password each time
- Escalate to Tier 2 if account shows suspicious activity
